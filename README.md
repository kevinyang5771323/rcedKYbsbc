# 项目简介

## 核心功能
- 用户答题
  - 提供每日挑战，用户回答一系列问题。
  - 问题涵盖不同类型，如一年级、二年级等。
  - 用户选择答案后，立即显示正确答案和解释。
- 积分系统
  - 用户每答对一题获得一定积分。
  - 积分用于升级用户等级，解锁新功能或奖励。
- 成就系统
  - 用户达到一定积分或完成特定挑战后获得成就。
  - 成就可以在个人资料中展示，增加用户的荣誉感。
- 排行榜
  - 显示用户在本地的排名。
  - 鼓励用户之间的健康竞争。

## 适用场景
- 数学学习
- 课后练习
- 自我提升

## 技术特点
- 采用Java开发，保证系统稳定性和性能。
- 微信小程序前端，方便用户随时随地使用。
- 界面简洁，操作便捷，用户体验良好。
